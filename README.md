# Курс рубля ЦБ РФ
### [Demo](https://ruble-rate.vercel.app/)

## О проекте:

- Приложение отображает текущий курс рубля по отношению к другим валютам
- Данные, используемые в приложении, предоставлены Центральным Банком РФ

## Технологии:

- Vue 3
- Vite
- Tailwind
- Axios
- Moment

## Установка:

npm install

## Запуск:

npm run dev
